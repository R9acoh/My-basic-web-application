How to Use This Tool

This Auto Marker Planning Software helps distribute marking workload among multiple markers.

  1. Enter your project name and deadline.
  2. Add markers with their expertise and daily capacity.
  3. Enter the total number of papers to be marked.
  4. Select an assignment strategy and click "Assign Papers".
